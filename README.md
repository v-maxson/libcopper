# Copper
A simple utility library for C99. This library is mostly for my own personal use. 
It is not intended to be a full-fledged library, nor is it intended to be the most efficient or optimized library.

The goal is to remain simple and easy to use, while still providing cross-platform support.

## Features
- [x] Cross-platform implementations for `asprintf` and `vasprintf` (dynamically allocate a string with `printf`-like syntax).
- [x] Dynamic Arrays (similar to C++ `std::vector`).
- [ ] Dynamic Strings (similar to C++ `std::string`).
- [x] Memory Arena (simple memory pool for fast allocation and deallocation).
- [ ] Logging.
- [ ] File I/O.
- [ ] Threading.
- [ ] Math.
- [ ] Time.
- [ ] RNG.
- [ ] Compression.
- [ ] Encryption.
